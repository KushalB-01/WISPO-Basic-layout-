# Wispo – Whisper of Support

Here is a frontend prototype of an anonymous mental health support platform.
It provides a simple and safe interface where users can select their emotions and express their thoughts through a chat system.

---

## Features

* Emotion selection (Sad, Anxious, Stressed, Lonely, etc.)
* Chat interface with message flow
* Rule-based responses (keyword-based)
* Smooth transitions and basic animations
* Responsive layout for desktop and mobile

---

## Tech Stack

* HTML
* CSS
* JavaScript

## How It Works

1. User opens the landing page
2. Clicks “Get Started”
3. Selects an emotion
4. Opens chat interface
5. Sends messages and receives predefined responses
